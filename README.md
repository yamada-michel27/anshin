# ANSIN

## 概要
このアプリは、友達との関係性を事前に登録し、
交際相手にワンタップで共有できる安心サポートアプリです。

「毎回説明するのが面倒」「伝え忘れて誤解される」といった悩みを減らし、
余計な不安やすれ違いを防ぐことを目的としています。

## 価値
    - 友人との関係性を毎回説明する手間を減らせる
    - ワンタップで関係性を共有でき、説明の負担を軽減できる
    - 関係性をアプリ上で可視化することで、伝え忘れや誤解を防げる
    - 不安やすれ違いの原因となる説明を減らし、安心感のある関係構築を支援する

## 技術
- Swift（iOSネイティブアプリ）
- 認証機能（ユーザー識別・データ保護）
